# 二十一回廊

一个静态网页端的回合制卡牌肉鸽21点变体原型。

## 当前规则

- 初始牌库为半副扑克牌：每局随机移除两个花色。
- 每回合先抽两张牌。
- 玩家可以继续摸牌或停牌。
- 停牌后扣除 `21 - 当前点数` 的筹码。
- 爆牌后扣除6枚筹码，符文可以改变结算。
- 筹码归零时游戏结束。
- 每次结算后获得金币并进入商店，再进入下一回合。
- 商店固定出现2个符文和2个画笔，金币可用于购买或刷新。
- 玩家最多持有3个符文，可在商店阶段出售符文。
- 抽牌堆和弃牌堆可以点击查看剩余所有牌。

## GitHub Pages

这个版本不需要构建步骤。把仓库发布到 GitHub Pages 后，入口文件就是仓库根目录的 `index.html`。
